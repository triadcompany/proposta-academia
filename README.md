# Proposta Comercial — DVTEAM Saúde e Performance

Landing page one-page (estilo pitch deck) com a proposta de serviços de marketing digital da **Triad Company** para a academia **DVTEAM Saúde e Performance**.

## Estrutura

- `index.html` — conteúdo e seções da página
- `styles.css` — identidade visual (cinza escuro + laranja, Poppins/Montserrat), layout responsivo
- `script.js` — scroll suave, barra de progresso e animações de entrada ao rolar

## Como visualizar

Sem build step necessário. Basta abrir `index.html` no navegador, ou servir a pasta com qualquer servidor estático:

```bash
python3 -m http.server 8000
```

## Publicar no GitHub Pages

1. Suba este repositório para o GitHub
2. Em *Settings → Pages*, selecione a branch principal e a raiz (`/`) como origem
3. A página ficará disponível em `https://<usuario>.github.io/<repo>/`
